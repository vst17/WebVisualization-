Summary: Latitude vs X

The purpose of this project is to analyze the weather for countries according to their distance to the equator. Using Jupyter Notebook and the OpenWeatherMap API I have retrieved 500+ random latitudes and longitudes. By using a Python library called Citipy, I was able to iterate through latitudes and longitudes to find the nearest city. Libraries such as Pandas and Matplotlib help generate a series of scatter plots to visually show the relationship between Latitude and other variables such as cloudiness, humidity,temperature, and wind speed.
